## 🧰 What this app does

offline-video-gallery-generator turns a folder of videos into a simple HTML gallery

You can open the gallery in your browser and browse your videos without uploading them anywhere

It is built for local use, so your files stay on your computer

## ✅ What you need

- A Windows PC
- A web browser such as Chrome, Edge, or Firefox
- Your video files in one folder
- PHP installed on your computer if the release does not include a ready-to-run package

## 📂 Before you start

Put the videos you want to view into one folder

Use a folder with clear file names if you can

Keep the folder in a place you can find again, such as Desktop or Documents

If your videos are in many folders, move the ones you want into one main folder first

## 🖥️ How to run on Windows

If the release includes an EXE file, double-click it to start

If the release includes a ZIP file, extract it first, then open the files inside

If the release uses PHP, make sure PHP is installed before you start

You can check PHP by opening Command Prompt and typing:

php -v

If you see a version number, PHP is ready

If Windows asks for permission, choose Yes

## 🧩 File types

The app is meant for common video files such as:

- MP4
- MOV
- MKV
- WEBM
- AVI

If a file plays in your browser, it should work well in the gallery

## 🛠️ Troubleshooting

If nothing happens when you open the app, check that you extracted all files

If the app does not start, make sure Windows did not block the file

If the gallery opens but videos do not play, try a different browser

If the folder looks empty, confirm that your video files are inside the folder you selected

If PHP is required and the app does not launch, install PHP and try again

## 🧪 Tips for best results

Use short file names with simple letters and numbers

Keep video files in one folder per gallery

Avoid very large folders if you want faster loading

Use a browser with current updates

Keep the output folder in the same place as the source folder unless the app tells you otherwise

## 🔐 Local use

The gallery runs on your computer

Your video files do not need to leave your device

This makes it useful for private collections and offline viewing
